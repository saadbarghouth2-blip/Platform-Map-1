# كنوز مصر — مشروع خرائط تفاعلية (React)

مشروع جاهز للوحدة الثالثة (كنوز مصر) للأطفال:
- خرائط تفاعلية (React Leaflet)
- طبقات Layers
- تقييم (MCQ) + سؤال "اضغط على المكان الصحيح"
- نقاط وتحفيز + حفظ التقدم في LocalStorage
