# كنوز مصر — مشروع خرائط تفاعلية (React)

مشروع جاهز للوحدة الثالثة (كنوز مصر) للأطفال:
- خرائط تفاعلية (React Leaflet)
- طبقات Layers
- تقييم (MCQ) + سؤال "اضغط على المكان الصحيح"
- نقاط وتحفيز + حفظ التقدم في LocalStorage

## التشغيل
1) افتح التيرمنال داخل مجلد المشروع:
```bash
npm install
npm run dev
```

2) افتح:
http://localhost:5173

## إضافة صوت/فيديو (اختياري)
- ضع ملفات صوت في `public/audio/` واستعملها داخل Popup لو حابب.
- أو ضع روابط فيديو داخل `info` أو اعمل زر "فيديو" بسهولة.

## تعديل المحتوى
المحتوى موجود في:
`src/data/lessons.js`

تقدر تضيف نقاط/أسئلة بسهولة.


> تم تحسين التصميم بإضافة framer-motion (للأنيميشن) + مؤثرات صوتية WebAudio.


## مميزات إضافية (نسخة الأطفال المتقدمة)
- أدوات رسم على الخريطة (Leaflet Draw)
- لعبة سحب وإفلات (react-dnd)
- نظام Levels (كل 50 نقطة)
- شهادة PDF باسم الطالب (jsPDF)
- فيديو داخل Popups (YouTube embed)

- Bottom Toolbar (استكشاف/رسم/قياس/قصة)
